slice:-

example:-
const data =[0,1,2,3,4]
data.slice(2,4)

result:-2,3


it will return from index 2 to 3 only it will not consider index 4
it will not change in original array

splice:-

example:-
const data = [0,1,2,3,4,5,6,7] data.splice(2,2) data.splice(2,3 ,12,13,14)

result:- 2,3


data.splice(2,2):-


it will remove from 2nd index to 2 element means (2 and 3) will from original array and return as answer

data.splice(2,3 ,12,13,14)


it will remove from 2nd index to 3 element means (2,3 and 4) will from original array and return as answer
but (12,13,14) will add in original array in place of remove array from index 2nd
it will change original array

switch:-


The switch statement is a part of JavaScript's "Conditional" Statements, which are used to perform different actions based on different conditions. Use switch to select one of many blocks of code to be executed. This is the perfect solution for long, nested if/else statements.
